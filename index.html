    A PIC16F84-Simulator developpen in Java

    Welcome to the Github.io Page from the TuxSim.
    TuxSim is a Simualor for the PIC16F84 and is developped in Java.

    For running a programm, you will need an assembled LST - File, the TuxSim is no assembler or compiler.

    Further you need a Java Runtime Environment (Java 7 recommended).
